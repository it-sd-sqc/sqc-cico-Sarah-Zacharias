# Module 6: Auto-submit Card Number Test

## Test steps
1. Build and run CiCo
2. Type in 8-digit card number (00000000)
3. Verify submission happens automatically after 8th digit
4. Verify Update button is no longer present

# Module 5: User Control and Freedom

## Test steps
1. Build and run CiCo.
2. Enter 00000000 into the field and click Update.
3. Click the "Next User" button.

## Expected results:
* The status panel should appear after step 2, showing "Checked IN" or "Checked OUT".
* Clicking the "Next User" button should end the current session and reset the system for the next user.
